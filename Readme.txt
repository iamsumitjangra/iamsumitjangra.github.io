# Sumit Kumar - Personal Portfolio Website

Welcome to the personal portfolio website of **Sumit Kumar** - a passionate traveller, painter, reader, and tennis enthusiast.

## 🌐 Live Website
**URL**: https://sumitsharma14k.github.io

## 📋 About This Project

This is a comprehensive personal portfolio website showcasing Sumit's diverse interests, projects, and experiences. The website features a clean, modern design with organized sections for different aspects of his life and work.

## 🎨 Design & Template

Built using the **iPortfolio Bootstrap Template** from BootstrapMade:
- **Template**: iPortfolio Bootstrap Portfolio Template
- **Source**: https://bootstrapmade.com/iportfolio-bootstrap-portfolio-websites-template/
- **License**: BootstrapMade License (see footer credits)
- **Customization**: Extensively customized and reorganized for personal use

## 🏗️ Website Structure

### Main Sections:
1. **Home** (`index.html`) - Landing page with hero section and navigation
2. **Projects** (`projects/`) - Technical projects and innovations
3. **Readings** (`readings/`) - Book collection and reading journey
4. **Resources** (`resources/`) - Technical resources and learning materials
5. **Travel Blogs** (`travel-blogs/`) - Travel experiences and adventures
6. **Paintings** (`paintings/`) - Art gallery and creative works

### Key Features:
- **Responsive Design**: Mobile-first approach with Bootstrap framework
- **Interactive Elements**: Smooth scrolling, animations, and hover effects
- **Social Media Integration**: Instagram and other social platforms
- **PDF Downloads**: Technical books and project documentation
- **Image Galleries**: Organized visual content with lazy loading
- **Consistent Branding**: Unified design language across all pages

## 🛠️ Technologies Used

### Frontend:
- **HTML5**: Semantic markup and structure
- **CSS3**: Modern styling with custom properties
- **JavaScript**: Interactive functionality and animations
- **Bootstrap 4**: Responsive grid system and components

### Libraries & Frameworks:
- **AOS (Animate On Scroll)**: Scroll animations
- **jQuery**: DOM manipulation and event handling
- **Owl Carousel**: Image carousels and sliders
- **Isotope**: Filtering and layout
- **Typed.js**: Typing animations
- **Venobox**: Lightbox functionality
- **Boxicons & Icofont**: Icon libraries

### Data Management:
- **CSV Files**: Structured data for books and travel blogs
- **JSON**: Configuration and metadata
- **Local Storage**: User preferences and settings
